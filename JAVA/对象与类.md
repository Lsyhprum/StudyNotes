# 对象与类

## 面向对象程序设计概述

传统结构化程序设计通过设计一系列的过程（即算法）求解问题，过程确定后则开始考虑存储数据的方式，即算法第一位，数据第二位。OOP 调换了次序。
对于一些规模较小的问题，比较适合分解为过程，而面向对象更适合解决大规模的问题。
