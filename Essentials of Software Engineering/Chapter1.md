# 软件工程概述

1. **软件**

软件是计算机系统中与硬件相互依存的另一部分，是包括程序，数据及相关文档的完整集合。

**Software = Program + Data + Document**

2. **软件危机**

    1. 问题：
        1. 如何**开发**软件，以满足对软件日益增长的需求
        2. 如何**维护**不断膨胀的软件
    
    2. 原因：
        1. 软件本身特点 —— 难于维护，逻辑复杂
        1. 软件 != 程序
        2. 无方法学，未摆脱手工生产方式

3. **软件工程**

    1. 定义：

    软件工程是应用计算机科学、数学及管理科学等原理开发软件的工程。它借鉴传统工程的原则、方法，以提高质量，降低成本为目的。

    2. 与计算机科学的区别：

    计算机科学侧重于理论研究，软件工程强调如何有效建造一个软件系统。
