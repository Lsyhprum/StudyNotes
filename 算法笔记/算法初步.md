# 排序

## 冒泡排序

```cpp
int A[maxn]; // 下标 0 - n-1
for(int i = 1; i < n; i ++){            // 趟数
    for(int j = 0; j < n - i; j ++){    // 遍历到当前剩余数最右边
        if(A[j] > A[j + 1]){
            swap(A[j], A[j+1]);
        }
    }
}
```

## 选择排序

```cpp
int A[maxn]; // 下标 1-n
for(int i = 1; i <= n; i ++){           // 趟数
    int k = i;
    for(int j = i; j <= n; j ++){       // 选出 [i, n] 中最小元素
        if(A[j] < A[k])
            k = j;
    }
    swap(i, k);
}
```

## 插入排序

```cpp
int A[maxn]; // 下标 1-n
for(int i = 2; i <= n; i ++){           // 趟数 n-1
    int tmp = A[i], j = i;
    while(j > i && tmp < A[j-1]){
        A[j] = A[j - 1];
        j --;
    }
    A[j] = tmp;
}
```
