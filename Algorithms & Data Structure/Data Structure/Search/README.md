# 基本二分查找

## 左闭右闭 / 左闭右开

需要注意的细节：

* 右侧为闭区间时：right = length - 1, 右侧为开区间时：right = length
* 为避免加法溢出：mid = left + (right - left) / 2
* (存疑) **左闭右闭循环结束后无需判断 left 所指元素是否等于所找元素**
  * [2, 2) -> [2, 3) -> [1, 2, 3)  
  * [2, 2) -> [1, 2) x
  * [2, 2) -> [1, 3) mid = 2
