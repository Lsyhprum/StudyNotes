#### const 限定符

被 const 限定符修饰的变量值不能被改变。默认情况下 const 变量不能被其他文件访问，若需要被其他文件访问，需要添加 extern 关键字。

把引用绑定到 const 对象上，称之为 **对常量的引用**，对常量的引用不能被用作修改它所绑定的对象。

```cpp
const int ci = 1024;
const int &r1 = ci;
r1 = 42; // 错误
int &r2 = ci;  // 错误，否则将允许 r2 改变引用对象的值
```

**指向常量的指针** 与 **对常量的引用** 类似，不能用于改变其所指对象的值。
